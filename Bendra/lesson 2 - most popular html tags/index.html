Most popular HTML elements

There are many HTML elements (also known as HTML tags), but not all of them are commonly used. We will run through most common ones, and shortly describe what are their use cases.

h1, h2, h3, h4, h5, h6 tags are used for headings. The h1 is the largest, and should be used to describe the main purpose of the page. For better Search Engine Optimization (SEO) you should only use a single h1 tag.

h2 tags are mostly used for section naming.
h3 tags are used for articles, posts and other page components headings
h4 - h6 tags are sub-headers

In page structure, heading tags should always be used ascendingly. You can't skip header size. Skipping a heading size, like : h1->h3 is a very bad practice, becouse SEO can't validate your page content properly.

Most common block elements

Block elements are mostly used as containers for other HTML elements. Their purpose is to suggest and describe content within them, most common are:

  • header - used for navbars, card headers, any block elements that represent the describing part of a component
  • section - used to describe one of many page sections
  • main - used for the main page section, which is the whole purpose of the page
  • article - post, article, or any self contained component
  • aside - used for sidebars, or any content which is on a side of a main page content
  • footer - used for page footers. Footers are mostly used to contacts, copyrights and page links
  • div - for any other purposes. This element has no special meaning. It is also a default HTML element

Lists

ul - unordered list

It is the most popular list. It is used for any general use lists. Although ul tag has some default styling, it is prefered by SEO to have common purpose components in a list. We can alter the style using CSS styles.

Fun part - media tags

Images

You can upload images to your page by using CSS, or using <img>. When using img tag you have to set up several attributes;

  • src - the path to the image which should be loaded
  • alt - text, which will be shown if image is not found by the src attribute
  • height - sets image height. Width is changed respectively by the image width to heigth ratio
  • width - sets image width. Height is changed respectively by the image width to heigth ratio
IMPORTANT! If both, height and width is specified, then image could get streched

Links - anchor tags

Links are elements, that can navigate you to a certain page section, or a different page.
